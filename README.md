# Objetivo
Desenvolver as páginas seguindo os layouts e estruturas apresentados nos exemplos do desafio, utilizando HTML, CSS e Javascript.

# Landing Page
A primeira página é uma Landing Page para uma loja na categoria de artigos esportivos, apresentando alguns de seus produtos e indicando as formas de pagamento.
As ofertas de produtos são carregadas através de um JSON local, simulando o consumo de uma API ou o fornecimento de conteúdo por um banco de dados. Cada um dos registros é disponibilizado dentro de um carrossel, constrúido com as classes do Bootstrap.
O layout elaborado é responsivo, partindo do mobile e servindo para diferentes tamanhos de tela.
Além do Bootstrap, foram utilizadas ferramentas como o Google Fonts (para fontes e tamanhos), a extensão Color Picker (para identificar cores na proposta) e o Figma (para tratamento de ícones e imagens).
# E-mail Marketing
A segunda etapa consiste em um template para e-mail marketing, mostrando em forma de grade os produtos que estão em promoção. Todos os produtos são clicáveis e apresentam um desconto de acordo com os dados obtidos pelo JSON, tornando as ofertas flexíveis.
